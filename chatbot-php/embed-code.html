üìä Analytics Integration

The widget automatically tracks events to Google Analytics if gtag is available:

  • chat_opened - When user opens the chat
  • chat_closed - When user closes the chat
  • message_sent - When user sends a message
  • bot_response_success - When bot responds successfully
  • bot_response_error - When bot encounters an error

üîç Troubleshooting

Widget Not Appearing?

  • Check that both CSS and JS files are loading correctly
  • Verify the file paths are correct relative to your website
  • Check browser console for JavaScript errors

Chat Not Responding?

  • Ensure the chatbot-php folder is in the correct location
  • Check that PHP is working on your server
  • Verify the OpenAI API key is configured in config.php
